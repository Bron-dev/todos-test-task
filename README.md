# Task Manager App

A React + TypeScript task management app with drag-and-drop, search, and bulk actions.

---

## Features

- Add and manage tasks and columns.
- Drag-and-drop tasks between columns.
- Search tasks with character highlighting.
- Mark tasks as complete/incomplete.
- Bulk actions: delete, mark complete/incomplete, move tasks.
- State is persisted in `localStorage`.

---

## Requirements

- Node.js >= 18
- npm >= 9 or Yarn >= 3

---

## Installation

1. Clone the repository:

```bash
  git clone https://github.com/Bron-dev/todos-test-task.git
  cd todos-test-task
```
2. Install dependencies:
```bash
  npm install
# or
  yarn install
```
3. Run development server:
```bash
  npm run dev
# or
  yarn dev
```

Open http://localhost:5173
in your browser.

Or you can check live version by following next link:
https://extended-todo.netlify.app/
